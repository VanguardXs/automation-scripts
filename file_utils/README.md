# File Utilities

Utilities for automating common file system tasks.

These scripts are focused on reducing repetitive manual work when handling large numbers of files.

## Scripts

### rename_files.py
Batch renaming files in a directory.

**Typical use cases**
- Adding prefixes or suffixes to filenames
- Removing unwanted characters
- Renaming files in bulk using a consistent pattern

The script is designed to be simple and safe to use for everyday automation tasks.
